Stopping agent attribution on commits (IDE + CLI)
-------------------------------------------------
• Cursor IDE: Settings → Agent → Attribution → off (commits and PRs from Agent).

• Cursor CLI: this repo’s .cursor/cli.json merges attribution off for sessions here.
