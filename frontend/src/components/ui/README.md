# AgroShield UI Component Library

A comprehensive, accessible, and responsive UI component library built with React, TypeScript, and Tailwind CSS.

## Features

- 🎨 **75+ Components** - Complete component library
- 🎯 **TypeScript Support** - Full type safety
- ♿ **Accessibility** - WCAG 2.1 AA compliant
- 📱 **Responsive Design** - Mobile-first approach
- 🎭 **Animations** - Smooth transitions and micro-interactions
- 🚀 **Performance** - Optimized for speed
- 📚 **Documentation** - Complete guides and examples

## Components

### Base Components
- Button, Input, Label, Select, Textarea
- Card, Avatar, Badge, Separator
- Alert, Dialog, Tooltip, Popover

### Navigation
- Breadcrumb, Menu, Menubar, Tabs
- Accordion, TreeView, Timeline

### Data Display
- Table, DataGrid, VirtualList
- Progress, ProgressRing, Skeleton
- Chart, Masonry, InfiniteScroll

### Forms
- Form, Field, Checkbox, Radio
- Switch, Slider, DatePicker
- FileUpload, ColorPicker

### Layout
- Container, Grid, Flexbox
- ScrollArea, Resizable, SplitPane

### Interactive
- Dropdown, Command, ContextMenu
- Tour, Rating, Chip, Kbd

### Advanced
- DragDrop, VirtualScroll
- TooltipGroup, AccordionGroup
- ProgressRing, EmptyState

## Usage

```tsx
import { Button, Card, Input } from '@/components/ui'

export default function Example() {
  return (
    <Card>
      <Input placeholder="Enter text" />
      <Button variant="primary">Submit</Button>
    </Card>
  )
}
```
